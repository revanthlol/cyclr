Cyclr Installation Guide

Firefox users can install Cyclr directly from the official Add-ons Store. For manual/unpacked installation, follow the steps below.

Chromium Installation Steps

  • Step 1 Extract the downloaded Chromium ZIP file to a folder on your system.
  • Step 2 Open the Chrome extensions manager page: 1. Go to chrome://extensions 2. In the top-right corner of that page, enable the Developer Mode toggle.
  • Step 3 Click the Load unpacked button in the top-left corner and select the extracted Cyclr folder.
Firefox Installation Steps (Manual)
  • Step 1 Extract the downloaded Firefox ZIP file to a folder on your system.
  • Step 2 Open the Firefox debugging page: 1. Go to about:debugging 2. Click This Firefox in the sidebar.
  • Step 3 Click the Load Temporary Add-on button and choose the manifest.json file inside your extracted Cyclr folder.

Once installed, press Alt + Q to start
