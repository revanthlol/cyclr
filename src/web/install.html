Install Cyclr

Follow these steps to manually load the unpacked extension in Chrome/Chromium or Firefox.

Chrome / Chromium-based

  • Step 1 Extract the downloaded ZIP file (double-click or extract all to a folder on your system).
  • Step 2 Open the Chrome extensions manager page: 1. Click to copy the extensions page URL. 2. Open a new browser tab. 3. Paste the URL (chrome://extensions) into the address bar. 4. In the top-right corner of that page, enable the Developer Mode toggle.
  • Step 3 Click the Load unpacked button in the top-left corner and select the extracted Cyclr folder.

Mozilla Firefox

  • Step 1 Extract the downloaded ZIP file to a folder on your system.
  • Step 2 Open the Firefox debugging page: 1. Click to copy the debugging page URL. 2. Open a new browser tab. 3. Paste the URL (about:debugging) into the address bar. 4. Select This Firefox from the left-hand sidebar menu.
  • Step 3 Click the Load Temporary Add-on... button and choose the manifest.json file inside your extracted Cyclr folder.

Once installed, press Alt + Q to start
