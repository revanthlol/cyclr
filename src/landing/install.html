Install Cyclr

Follow these steps to manually load the unpacked extension in Chrome or any Chromium-based browser.

  • Step 1 Extract the downloaded ZIP file (double-click or extract all to a folder on your system).
  • Step 2 Open the Chrome extensions manager page by navigating to chrome://extensions. Enable the **Developer Mode** toggle in the top-right corner.
  • Step 3 Click the **Load unpacked** button in the top-left corner and select the extracted Cyclr folder.

Once installed, press Alt + Q to start
